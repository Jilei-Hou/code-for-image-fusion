This software release consists of an implementation of the algorithm described in the paper:

B. K. Shreyamsha Kumar, “Multifocus and Multispectral Image Fusion based on Pixel Significance using Discrete Cosine Harmonic Wavelet Transform”, Signal, Image and Video Processing, 2012. (doi: 10.1007/s11760-012-0361-x)


Download Original manuscript from http://link.springer.com/article/10.1007%2Fs11760-012-0361-x


Download draft manuscript from https://sites.google.com/site/shreyamsha in the Publications link.


It consists of the following files:

readme.txt: this file

images used: disk512, gun, kayak, med256

wt_fusion_pgm: main program

support .m files: dchwt_fn2, fusion_perform_fn,objective_fusion_perform_fn, imhist_fn, joint_hist_fn, method2_sivip2011_fn, per_extn_im_fn, sobel_fn

If you are using the modified Fusion Artifacts (NABF) measure, modification of Petrovic's Fusion Artifacts measure, please cite:

B. K. Shreyamsha Kumar, “Multifocus and Multispectral Image Fusion based on Pixel Significance using Discrete Cosine Harmonic Wavelet Transform”, Signal, Image and Video Processing, 2012, doi: 10.1007/s11760-012-0361-x
